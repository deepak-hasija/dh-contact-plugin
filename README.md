# DH Contact Plugin

A custom WordPress plugin built for professional contact 
form management with AJAX submission, admin dashboard, 
and CSV export functionality.

## Features
- AJAX form handling — no page reload
- Admin dashboard to view and manage entries
- CSV export of all submissions
- Unread count indicator in WordPress admin bar
- Settings page for configuration

## Installation
1. Upload plugin folder to /wp-content/plugins/
2. Activate through WordPress Plugins menu
3. Configure via Settings > DH Contact

## Requirements
- WordPress 5.0+
- PHP 7.4+
